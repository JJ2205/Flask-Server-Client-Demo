This project demonstrates a simple yet powerful REST API built using **Flask**, supporting all major HTTP methods ‚Äî `GET`, `POST`, `PUT`, `PATCH`, and `DELETE` ‚Äî on two endpoints: `/greet` and `/echo`.

It includes both a server (`Flask`) and a client (`requests`) to test and simulate interactions.


---

## üöÄ Features

### `/greet` endpoint
- **GET**: Returns a greeting message
- **POST**: Accepts JSON data and echoes a confirmation
- **PUT**: Replaces the current greeting message
- **PATCH**: Appends additional content to the greeting
- **DELETE**: Clears the greeting message

### `/echo` endpoint
- **GET**: Responds with a generic GET message
- **POST**: Echoes back the sent JSON data
- **PUT**: Echoes back a full update
- **PATCH**: Echoes back a partial update
- **DELETE**: Acknowledges the deletion
